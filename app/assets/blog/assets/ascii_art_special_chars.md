# MONOSPACED FONT
	a fixed-width (non-proportional) font, is a font whose letters each occupy the same amount of space.
	This contrasts to variable-width fonts, where the letters differ in size to one another.
	The first monospaced typefaces were designed for typewriters, which could only move the same distance forward with each letter typed.
	Free monospace fonts: Terminus, Anonymous, Anonymous Pro, Envy Code R, Source Code Pro, Inconsolata, M+ Fonts, Fixedsys Excelsior, Droid Sans Mono

# On a typewriter, a new line is carried out by two actions.
	The print position goes to the beginning of the line (Carriage Return)
	The print position moves and down to the next line (Line Feed)

	Windows text files 	use \r\n to terminate lines
	UNIX text files 	use \n   to terminate lines
