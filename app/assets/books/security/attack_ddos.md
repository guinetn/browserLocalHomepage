## Attack DDoS

overwhelming a victim with data/request flow to make it unavailable

    Internet
    
        Attacker 
            ↓
        send instructions to bot
            ↓
        Botnet (group of bot)
            Zombies/bots: infected machine
                ↓
                ↓  send many ip requests to override target capacity
                ↓
              Target

              Symptoms: 
                slow, not available, high volume from unic IP/IP range
                more requests on a single page
                user with same profil behavior (device, geolocalisation)
