## UUID

https://en.wikipedia.org/wiki/Universally_unique_identifier
the most used universal identifiers in software development
uniqueness: UUID duplicated probability is not zero but close enough to zero to be negligible

16 octets = 32 hexadecimal (base-16) digits
    123e4567-e89b-12d3-a456-426614174000          36 characters: 32 hexadecimal characters + 4 hyphens
    xxxxxxxx-xxxx-Mxxx-Nxxx-xxxxxxxxxxxx

The four-bit M and the 1 to 3 bit N fields code the format of the UUID itself.

128-bit label used for information in computer systems

 GUID: globally unique identifier
