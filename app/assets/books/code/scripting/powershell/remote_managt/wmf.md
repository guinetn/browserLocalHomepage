# WMF - Windows Management Framework

Management interface for Windows
To manage various versions of Windows client/Server

- Windows PowerShell - PowerShell Core?
- Windows PowerShell Desired State Configuration (DSC): configuration as code 
- Windows PowerShell Integrated Script Environment (ISE): development environment
- Windows Remote Management (WinRM)
- Windows Management Instrumentation (WMI)
- Windows PowerShell Web Services (Management OData IIS Extension)
- Software Inventory Logging (SIL)
- Server Manager CIM Provider
