# WinRM - Windows Remote Management

PowerShell distant communication
- version 2.0: WS-Man protocol = « Windows Remote Management » 
WinRM = Microsoft WS-Man implémentation 

Will be replaced by ssh

WinRM goes up with Windows Server 2003 R2
Allow to send management requests to machines via TCP/IP on ports 5985 (HTTP) / 5986 (HTTPS) using WS-Management protocol that is a DMTF specification (Distributed Management Task Force), defining a communication protocol tp manage servers, equipments, applications based on Web Services (SOAP)
