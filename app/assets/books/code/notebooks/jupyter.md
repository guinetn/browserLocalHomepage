## Anaconda

Install: https://www.anaconda.com/distribution

### Anaconda Prompt

Check Jupyter is installed+present on the path:
> jupyter kernelspec list
    Available kernels:
    .net-csharp    C:\Users\nguin\AppData\Roaming\jupyter\kernels\.net-csharp
    .net-fsharp    C:\Users\nguin\AppData\Roaming\jupyter\kernels\.net-fsharp
    julia-1.5      C:\Users\nguin\AppData\Roaming\jupyter\kernels\julia-1.5
    python3        C:\Users\nguin\anaconda3\share\jupyter\kernels\python3
    
Start jupyter from your Anaconda Prompt:
>jupyter lab
>jupyter notebook 
From browser create notebooks using Pytho, C#, F#, PowerShell
