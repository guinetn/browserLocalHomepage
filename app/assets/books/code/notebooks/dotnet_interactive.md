# .NET INTERACTIVE

Avoid to create a console to quick benchmarking your code 

1. Install the extension (don't need to install Python or Anaconda)
2. To open an existing .NET notebook
Ctrl+Shift+P → .NET Interactive: Open notebook → .ipynb
