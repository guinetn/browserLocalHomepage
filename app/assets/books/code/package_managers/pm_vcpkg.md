# VCPKG

C/C++ dependency manager from Microsoft
For all platforms, buildsystems, and workflows
manage C and C++ libraries on Windows, Linux and MacOS
vcpkg is a cross-platform open source package manager by Microsoft
announced at CppCon 2016
supports Visual Studio 2015 Update 3 and above

- https://vcpkg.io/en/
- https://github.com/microsoft/vcpkg


.\vcpkg search sqlite
.\vcpkg install sqlite3
.\vcpkg install sqlite3:x86-uwp zlib:x64-windows
.\vcpkg list
