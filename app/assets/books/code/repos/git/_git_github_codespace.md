## GITHUB CODESPACE - GHC

Codespaces are an online implementation of Visual Studio Code.
Editeur basé sur un navigateur qui prend en charge les référentiels Git, les extensions et une interface de ligne de commande intégrée qui vous permet de modifier, d’exécuter et de déboguer vos applications depuis n’importe quel appareil.

Develop entirely in the cloud using Codespaces, an integrated development environment (IDE) on GitHub.
you will be billed for storage and compute usage.
Connect to Codespaces from 
	Visual Studio Code
	Visual Studio 
	a browser-based editor

How does it work?
	Codespaces are built on top of another existing Microsoft solution called Visual Studio Codespaces (VSC), 
	which is an excellent paid alternative to GH Codespaces when you are looking for something more open, this 
	is because VS Codespaces create a virtual machine on Azure and connect to it through the native 
	VSCode functionality called Remote Development.

use VS Codespaces to create a remote development environment
	"Create Codespace" button → series of awesome options that can be toggled
	dotfiles, which are the configuration files for our shell.
