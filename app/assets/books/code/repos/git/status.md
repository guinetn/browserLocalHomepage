## git status

Show working directory and staging area files status (staged, unstaged, untracked)
To see wich changes staged, which haven’t, which files aren’t tracked
what's been going on with git add and git commit
Checks the status of a given repository. Lists the working branch and any new/changed files that need to be committed.

untracked files have 
	. just been added to the project and haven't been committed yet
	. are listed in .gitignore file (binaries...)
	Git ignore files listed in the '.gitignore' file (on a separate line, * as a wildcard)
