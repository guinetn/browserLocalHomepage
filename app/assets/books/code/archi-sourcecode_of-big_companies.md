# ARCHITECTURE OF BIG COMPANIES

https://lnkd.in/eSi8Efa

Smart Accounting is solution built with Azure cloud services and microservices architecture. You can upload invoice scan and store invoice data in the database together with original file.
I decided to use Docker and Kubernetes but the whole solution could be built using serverless components too.
Some of the concepts you can find in this project:
1. How to implement asynchronous communication with Azure Service Bus
2. How to run microservices with Docker
3. How to store data in the Azure Cosmos DB
4. How to user Form Recognizer to scan file content
5. How to store files on Azure Blob Storage
6. How to authenticate users and implement authorization with Azure Active Directory B2C
7. How to use Azure SQL database to store event logs
8. How to use Blazor to implement end users web app
