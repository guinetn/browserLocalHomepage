## RFC - Request for Comments

https://www.rfc-editor.org

Technical document published by the Internet Engineering Task Force (IETF) describing methods, behaviors, research, or innovations applicable to the working of the Internet.
Used to develop 
- a "standard" network protocol
- a function of a network protocol
- any feature which is related with network communication

Over 8 650 RFCs (2019)

### How to write an RFC?
- Copy the RFC template and write your proposal! ...
- Start a [DISCUSS] thread on the mailing list and link to the proposal's wiki page. ...
- Answer questions and concerns on the email thread. ...
- After the deadline for feedback has been reached summarize the consensus and your decision on the [DISCUSS] thread.
