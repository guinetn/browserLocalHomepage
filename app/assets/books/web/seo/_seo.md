Le référencement payant qui consiste à payer pour que le site apparaisse en première page du moteur de recherche et le référencement naturel qui consiste à faire en sorte que le moteur de recherche sélectionne le site pour le mettre en première page.
Le référencement naturel peut-être définit comme un ensemble de méthodes permettant de positionner un site en tête des résultats de recherche d’un moteur de recherche.
Le référencement payant repose quant à lui sur la mise aux enchères des positions de recherches les plus visibles sur des demandes ou questions précises avec des liens dits « sponsorisés ».

La mission du SEO pour Search Engine Optimisation consiste à analyser les pages d’un site, indiquer, et parfois réaliser les démarches nécessaires pour positionner le contenu du site en haut des pages de recherche de certains mots clés spécifiques.

## Les critères On-page d’optimisation

Les moteurs de recherche veulent faire leur travail le mieux possible en se référant aux utilisateurs de sites Web et le contenu qui est le plus pertinent à ce que l’utilisateur recherche comment déterminée la pertinence?

Contenu: est déterminée par le thème qui est donné, le texte, les titres et les descriptions qui sont donnés ( un paragraphe Anti–spinning pour réduire le pourcentage de duplicate content dans le texte )
Performance: la vitesse de votre site et le temps de chargement et le poids de la page web (Page speed Insights)
Les mots clés : utilisation naturelle des mots clés et éviter la sur-optimisation dans l’article et éviter d’augmenter la densité de mots clés
Autorité: votre site a un contenu de qualité suffisante (pas de duplicate content) pour lier ou faire d’autres sites faisant autorité utilisent votre site comme une référence ou citent l’information qui est disponible.
Construire une stratégie des liens internes : construire son maillage interne ( une bonne netlinkg )
Utilisez les technique de rédaction pour le web :
un paragraphe anti-spinning pour diminuer le pourcentage de duplicate content et la sur-optimisation dans la page
utilisation de la longue traîne
utiliser la recherche associée pour choisir la longue traîne
Meta keywords : sont à peu près ignorés par les moteurs de recherche de nos jours, mais si vous utilisez encore eux, assurez-vous qu’il parle spécifiquement à cette page et qu’il est aussi formaté correctement
Meta description : Votre méta description doit être unique et parler aussi de cette page spécifique. ( La balise méta description c’est concernée pour l’internaute et pas pour le SEO )
Meta title : La balises de titre devraient également être unique! Vous pensez que votre titre comme un 4-8 mot annonce, si faire de votre mieux pour inciter le lecteur alors ils veulent et cliquez sur lire la suite. (La balise doit contenir 11 mots et 70 caractères)
