# Dotnet Standard: unity

https://github.com/dotnet/standard
https://github.com/dotnet/standard/tree/master/docs/versions

- Solves .Net code sharing across platforms by bringing all the APIs that you expect
- unifies .NET platforms and prevents future fragmentation
- A set of APIs that all .NET platforms have to implement 
- .NET Standard 2.0 will be implemented by .NET Framework, .NET Core, and Xamarin. 
- Replace Portable Class Libraries (PCLs) as the tooling story for building multi-platform .NET libraries.
