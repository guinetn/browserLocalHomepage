## DLL - Dynamic Link Library

dll = shared libraries in UNIX-based operating systems
- to share code and resources
- to shrink the size of your apps.
- to extend your app

The DLL uses the C calling convention
It can be called from apps written in other programming languages, as long as the platform, calling conventions, and linking conventions match
