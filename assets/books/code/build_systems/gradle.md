Build system for the JVM
Gradle helps teams build, automate and deliver better software, faster.
From mobile apps to microservices, from small startups to big enterprises

Build Anything
Write in Java, C++, Python or your language of choice.
Package for deployment on any platform. Go monorepo or multi-repo. 
And rely on Gradle's unparalleled versatility to build it all.

Automate Everything
Use Gradle's rich API and mature ecosystem of plugins and integrations to get ambitious about automation. 
Model, integrate and systematize the delivery of your software from end to end.

Deliver Faster
Scale out development with elegant, blazing-fast builds. From compile avoidance to advanced caching and beyond, we pursue performance relentlessly so your team can deliver continuously.

Install
> choco install gradle
gradle -v

Under System Variables select Path, then click Edit. Add an entry for C:\Gradle\gradle-6.5\bin

SDKMAN! is a tool for managing parallel versions of multiple Software Development Kits on most Unix-based systems.
$ sdk install gradle 6.5

Homebrew is “the missing package manager for macOS”.
$ brew install gradle
