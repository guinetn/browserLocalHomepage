# AdonisJS

https://preview.adonisjs.com/

When you are building an application, you need more features than a routing system. You need to:
- access a database
- validate your data
- create an authentication layer
- create an authorization system

AdonisJS = No need to download a gazillion of packages to create a meaningful application. AdonisJS comes with ton of features for every corner of your app, aiming to make you productive from day one.

HTTP: Router, form validation, JSON Serializers, Template Engine (Edge)
DATABASE: SQL First, ORM, Migrations, seeds 
AUTH: Multi Driver Auth, RBAC Role based access control is baked into the framework, providing you the fine grained API for implementing authorization across the application
SECURITY: CSRF Protection, Web Shield, CORS Enabled
MUCH MORE: Health Checks
