## Google free tier
Google may charge if you go over certain "free tier" limits etc

Free Virtual Private Server instance on the Google Compute Engine
1. https://console.cloud.google.com/ - sign up/in
2. https://console.cloud.google.com/compute - top left hamburger menu 
3. Enable Compute Engine if needed, then set up billing
4. Create new instance in one of the free tier locations (Iowa, Oregon, South Carolina) with machine type f1-micro.
5. Finish the server config (Disk...)
6. Click [Connect via SSH button] to log in
