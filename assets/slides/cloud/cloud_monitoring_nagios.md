## Nagios

Open-source tool used for system-wide monitoring (systems, services, networks and infrastructure)
Helps System/DevOps Admin to monitor hosts and services of those hosts in a Datacenter or AWS environment.

You can observe any number of devices as long as they are connected to the internet and have the Nagios plugin installed. If there is not a plugin available for your device, we can code our own using various coding languages.  
Graphical user interface to configure Nagios with few clicks, and start monitoring your devices right away. https://www.nagios.com/products/nagios-xi/#features
