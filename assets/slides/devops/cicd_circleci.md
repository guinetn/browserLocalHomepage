# Circle CI

https://circleci.com/

one of the best CI/CD tools that is suited for realizing CI/CD for open-source projects and large-scale projects. CircleCI Cloud is its cloud-based offering whereas CircleCI Server is its on-premise (or self-hosted) solution.
It supports languages that can be built on Windows, Linux, and macOS platforms. It is easy to set up and uses a proprietary YAML syntax for its pipelines.
