# GEODATA

## Geocoding 
Converts an address/place name → latitude/longitude   
Accuracy = f(provider)
    
## Reverse geocoding
Converts geographic coordinates → a location description (place name, addressable location)

* Providers (API Key or other parameters needed)
- Google Maps
- Mapquest
- HERE Maps (arguably the most comprehensive.)
- OpenStreetMap: https://wiki.openstreetmap.org/wiki/About_OpenStreetMap (No API Key)
